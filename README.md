# 🤖 نظام التداول الذكي - Trading AI System

نظام تداول متقدم يعمل بالذكاء الاصطناعي مع تحليل تلقائي 24/7 وتنبيهات Telegram.

## ✨ المزايا الرئيسية

### 🧠 الذكاء الاصطناعي
- **المحلل الذكي**: تحليل فني وأساسي شامل باستخدام Gemini AI
- **المقيم الذكي**: تقييم دقة الإشارات وتصنيفها
- **المحسّن الذكي**: اقتراحات لتحسين الأداء

### 📊 التداول
- **المحاكي الذكي**: محاكاة واقعية للتداول بدون مخاطر
- **التداول الحقيقي**: اتصال مباشر مع Alpaca (Paper/Live Trading)
- **إدارة المخاطر**: حدود تلقائية للحماية من الخسائر

### 🔄 التشغيل التلقائي
- **GitHub Actions**: تحليل تلقائي كل 4 ساعات
- **تنبيهات Telegram**: إشعارات فورية للفرص القوية
- **تقارير يومية**: ملخص شامل يومياً الساعة 9 صباحاً

### 💾 حفظ البيانات
- **localStorage**: حفظ تلقائي للمحفظة والصفقات
- **Supabase** (اختياري): قاعدة بيانات سحابية
- **لا تفقد بياناتك**: كل شيء محفوظ حتى بعد إغلاق المتصفح

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+
- pnpm (أو npm)
- Gemini API Key
- Telegram Bot Token (للتنبيهات)

### التثبيت

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/trading-system.git
cd trading-system

# Install dependencies
pnpm install

# Create .env file
cp .env.example .env

# Add your API keys to .env
# GEMINI_API_KEY=your_key_here
# TELEGRAM_BOT_TOKEN=your_token_here
# TELEGRAM_CHAT_ID=your_chat_id_here

# Start development server
pnpm run dev
```

## 📱 استخدام النظام

### 1. المحلل الذكي
- أدخل رمز السهم (مثل: AAPL)
- أو حلل قائمة المتابعة كاملة
- احصل على تحليل شامل مع توصيات

### 2. المقيم الذكي
- يستقبل الإشارات من المحلل تلقائياً
- يقيّم دقة كل إشارة
- يرسل الإشارات الموثوقة للمحاكي والتداول الحقيقي

### 3. المحاكي الذكي
- يستقبل الإشارات تلقائياً
- ينفذ الصفقات فوراً
- تراقب الأداء والأرباح
- كل شيء محفوظ في localStorage

### 4. التداول الحقيقي
- اتصل بحساب Alpaca
- فعّل التداول الحقيقي
- النظام ينفذ الصفقات تلقائياً
- إعدادات إدارة المخاطر محفوظة
